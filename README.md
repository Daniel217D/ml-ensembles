# ML Ensembles

Учебный проект по тестированию ансамблей моделей при обучении. Репозиторий собирает отдельные ноутбуки с экспериментами по bagging, random forest, boosting и другим ансамблевым подходам.

## Что внутри

Проект нужен для практики с ансамблевыми подходами и проверки их эффекта на качестве модели:

- бутстрап-выборки для каждого базового алгоритма;
- сравнение одиночных моделей и ансамблей;
- анализ качества на разных наборах данных;
- проверка влияния гиперпараметров на итоговые метрики.

## Быстрый старт

1. Клонируйте проект.
2. Создайте виртуальное окружение и установите зависимости:

```bash
python -m venv .venv
source .venv/bin/activate   # или .venv\Scripts\activate в Windows
pip install -r requirements.txt
```

3. Запустите Jupyter:

```bash
jupyter notebook
```

4. Откройте нужный `.ipynb` и выполните ячейки по порядку.
